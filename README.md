# 🇧🇷 BrasilMatch - Dating App para Brasileiros no Exterior

App de relacionamento focado em conectar brasileiros vivendo nos Estados Unidos.

## 🎨 Design
- **Paleta:** Sunset Romance (coral + laranja + dourado)
- **Estilo:** Divertido Casual
- **Foco:** Relacionamentos sérios

## 🛠️ Tech Stack

### Frontend (Mobile)
- **Framework:** Flutter 3.16+
- **Linguagem:** Dart 3.0+
- **State Management:** Riverpod 2.4+
- **Routing:** go_router

### Backend
- **Database:** Supabase (PostgreSQL)
- **Auth:** Supabase Auth
- **Storage:** Supabase Storage
- **Real-time:** Supabase Realtime
- **Admin Panel:** Next.js (Vercel)

### Monetização
- **Modelo:** Créditos (consumíveis)
- **Preços:** $2.99 (20 créditos), $5.99 (50 créditos), $9.99 (120 créditos)
- **IAP:** Apple In-App Purchase + Google Play Billing

## 📂 Estrutura do Projeto

```
brasilmatch/
├── flutter_app/          # App mobile Flutter
├── supabase/            # Database schemas & migrations
├── backend/             # Admin panel (Next.js)
├── scripts/             # Automação (git pull, wireless debug)
└── docs/                # Documentação setup
```
